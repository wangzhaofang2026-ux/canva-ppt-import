项目定位

云台山的不可复制资源

云台山不是背景，而是这台演出的根

PART TWO

第二部分

故事创意

这台演出讲什么，为什么适合云台山？

精神内核

千百年来，人们走进云台山，本质都是在寻找内心的一份安顿。

云台山的五条文脉，天然指向同一个精神内核：无论来自何方，走进这片山水，总能在某一刻与自己和解。

核心立意

山水相逢，心安云台。

《问水云台》正是将这千年修心传统，转化为一个当代人也能共鸣的故事：无论你从哪里来，走进这片山水，总能在某一刻，与自己和解。

山水之间，最美的不是风景，是相逢。相逢之后，最好的不是留下，是一起走过一段路，然后各自带着力量回家。

剧情简介

摆渡人云川误入云台秘境，在药伯与山阳七翁的指引下唤醒灵石、救回水脉灵体灵汐，与众人共同修复山林灵脉。

一路历险、相逢、飞渡、守光、重生——云川在帮助秘境回归秩序的同时，也放下了心中焦躁，寻回了久违的安宁。

一则关于山水、相逢、守护与归心的东方寓言。

PART THREE

第三部分

创编理念

云台山文化如何转化为剧目？

核心原则

我们不做文化说教，不演历史故事，不设一句台词。

全剧以"润物细无声"的方式，让文化化为剧目底色。观众不是来"受教育"的，而是在杂技的惊险与山水的治愈中，"亲历一段旅程"。

三个核心原则：

1、文化不讲解——符号化入人物与场景，观众感受到即可

2、山水不背景——将云台实景转化为演出场景，景随戏变

3、故事不独白——无台词，杂技、舞台、光影、特效、音乐共同推动叙事

文化不讲解：人物的文化转化

云台山千年文脉，不设讲解员、不写字幕卡，化为角色的底色：

王维"登高修心" → 主角云川的精神弧光：从焦躁走向从容

竹林七贤"隐逸风骨" → 山阳七翁：七人群体，以"七"应"七贤"

药王孙思邈"采药济世" → 药伯：身背药筐，一路引路救人

真武大帝"修行发源" → 真武无相道力：无形之风，驱动一切"巧合"

云台山水 → 转化为灵汐、水纹云笛、生命之树、溪瀑云雾等舞台意象

人物形象设定

云川｜摆渡人

灵汐｜水脉灵体

药伯｜引路人

山阳七翁｜守山者

浊心石｜危机核心

不是简单的反派，而是执念、迷茫与浊气的凝结

山水不背景：景区实景 → 演出场景

景区的真实地貌成为剧情发生的空间和情绪变化的载体。

茱萸峰 → 云端守望与日出

红石峡 → 峡谷惊澜与冒险

潭瀑峡 → 灵溪与生命之树

天瀑 → 空间打开与高潮

子房湖 → 归心与安宁

故事不独白：融合推动叙事

不用台词推动剧情，而用杂技、音乐、光影、特效和身体动作完成叙事。

每一个节目的设定，都必须回答"为什么在这里发生"

高台跳水 = 船翻落水的惊险与挣扎

大飞人 = 飞跃天瀑、打开云台空间的高潮动作

环球飞车渐缓 = 山河归序的安宁与余韵

剧情动机 + 杂技难度 + 空间奇观 + 情绪推进 = 真正的沉浸式山水杂技秀

双线叙事结构 · 七段式情绪旅程

全剧由序章、五幕正剧、终章构成，采用外部行动线与内部心路线双线交织推进：

七幕情绪递进：

静美入画 → 惊奇入境 → 温暖相逢 → 扶摇开阔 → 同心坚守 → 热血澎湃 → 山水归宁

七幕剧情结构：

缘起云台 → 峡谷惊澜 → 灵汐醒脉 → 逆流飞渡 → 风起定浊 → 山河澄明 → 万物归心

PART FOUR

第四部分

内容设计

观众每一幕会看到什么、感受到什么？

序章 · 缘起云台

场景：茱萸峰晨光、红石峡丹壁、潭瀑峡水声、天瀑暗影，构成云台山水明净、奇妙的静美画卷。将观众第一次拉入云台秘境。

观众体验：静美 · 期待

从第一秒开始，观众便不再是旁观者，而是即将踏入秘境的旅人。

第一幕 · 峡谷惊澜

场景：第一幕用湖面大雾、翻船落水、悬索高空、崖壁救援制造第一个强刺激点。

观众体验：从屏息到惊呼，从危机到希望。

核心节目：高台跳水、浪桥入水

当观众以为这只是一场风景时，危险已经降临。

第二幕 · 灵汐醒脉

场景：第二幕以灵溪脱困、生命之树复苏和手技趣味，与观众建立温暖与共鸣的情感连接。

观众体验：从惊险惊奇 → 温暖欢乐

核心节目：手技、绸吊、皮条

危机之后，是温暖的相逢。灵汐醒来，观众的心也随之柔软。

第三幕 · 逆流飞渡

场景：天瀑之下，水声轰鸣。通过大飞人打开云台空间，形成全剧第一次真正的开阔感。

观众体验：温暖汇聚 → 空间打开、肾上腺素提升、希望重升。

核心节目：大飞人、荡绳

当大飞人穿越天瀑，云台山的空间真正为观众打开了。

第四幕 · 风起定浊

场景：观众进入由180度环绕威亚、浪桥入水构成的微光被围、三力相守的至暗时刻，但微光不灭。

观众体验：不是悲壮压抑，而是同心、守光、坚持。

核心节目：威亚群舞、浪桥入水

至暗时刻，三力相守。观众不是在看戏，是在一起守光。

第五幕 · 山河澄明

场景：天瀑重现、光尘逆流、飞轮旋舞，山河回春。

观众体验：热血澎湃、光芒爆发、全场情绪登顶。

核心节目：四轴飞轮、环球飞车

当飞轮在光影中旋转，山河澄明，所有情绪在此刻爆发。

终章 · 万物归心

场景：茱萸峰云海、子房湖意象、云雾清光、环球飞车，共同构成山水归宁的画卷。

观众体验：从热血澎湃，回归温暖留恋、余韵悠长。

核心节目：环球飞车渐缓、集体造型

演出结束，但云台山的山水已在心中。观众带着力量回家。

一场连续的视听奇观

不是"看剧情"，而是进入一场连续的视听奇观。

每一幕都有视觉记忆点，每一段节目都有情绪落点。

从静美入画到山水归宁，70分钟的旅程如同一场电影——

有开篇的悬念，有中段的高潮，有结尾的留白。

PART FIVE

第五部分

舞台与空间系统

我们如何让故事真正发生在观众眼前？

舞台创意：水陆空四维立体表演体系

水系统

主舞台大型可蓄水区域，浅水/中水/深水分级配置，配合高空喷水、水幕、瀑布等多形态水效。

空中系统

多套高速特技威亚，三维空间移动，承载大飞人、皮条、绸吊等空中节目。

水陆转换

水下装置升起变为旱地舞台，水面平台平移转换为水陆两栖表演区。

舞台不是固定布景，而是会呼吸、会变化、会参与剧情的山水生命体。

视觉创意：山水入戏

180度环绕影像系统：观众被画面包裹，从"看景"到"在景中"

270度全域灯光系统：光即心境，清蓝暖白对应澄明，暗红墨绿对应困局

水面投影mapping：溪流纹理、潭水倒影直接投射在水面，演员行走在溪流之上

高空投影：瀑布倾泻，虚实交织

实景为骨、影像为魂——让云台山的风骨在舞台上"活"起来。

场景转换系统：让场景在观众眼前生长出来

为了保证演出连续性，本剧不依赖黑场换景，而是通过机械、水效、影像、灯光、雾效的同步联动完成场景变化。

核心转换方式：

水下平台升降、水陆舞台切换、中区LED开合、高空威亚移动、雾森与干冰造景、水幕喷泉飞瀑联动、灯光与音乐同步切换

峡谷可以在雾中合拢，溪流可以在脚下流动，天瀑可以在空中倾泻，云海可以在终章铺开。

服装创意：山水为骨，入画成衣

演员不"穿"山水，而是成为山水本身。

三种服装语言：

"立"如山岩：表现守护、力量与风骨

"流"若云水：表现灵动、柔韧与生命

"融"似烟岚：表现归山、归水、归心

色彩体系：玉脂白、水墨黑为基底；青花蓝、国槐绿为过渡；琉璃黄、中国红为克制高光

音乐创意：专属声音指纹

让旋律代替台词说话，让每一寸舞台有专属的声音指纹。

声音系统：

多声道环绕音箱、分区声场控制，水面、高空、观众区差异化声场，音乐、音效、特技声、环境声精准联动。

音乐方向：

融合国际化配乐制作思路与中原国风基调，提取太行山水、山阳古调、民乐音色，形成云台专属声音。

观众闭上眼睛，也能听见云台山的峡、瀑、雾、峰。

安全保障：让震撼不留遗憾

安全不是表演的枷锁，而是高难度表演敢于成立的底气。

演员安全

高空威亚双路备份、水下通讯与应急供气、湿滑舞台防滑处理、演出用水温度控制、每日设备检测与安全交底

设备安全

升降系统备用动力、机械锁止与位置传感、威亚电机冗余电源、LED与投影备份信号、总控平台故障报警

观众安全：疏散动线设计、观众区应急照明、人流监测与分流引导、互动区域安全隔离、极端天气预案

所有惊险，都有预案兜底。演员放心演，观众放心看。

PART SIX

第六部分

核心亮点

这台剧的独特价值是什么

文化强绑定——只能云台山

国内山水演艺多为歌舞或光影秀，《问水云台》以杂技特技为核心呈现方式，将真武修行、七贤隐逸、药王采药、王维登高五条文脉转化为人物气质与场景底色。

这台剧离开云台山，故事就不成立；离开云台山水，视觉就不成立；离开云台文脉，精神就不成立。

云台山独有的文化富矿 + 不可复制的山水地貌 = 这场演出的绝对唯一性。

杂技强支撑——顶级艺术创作团队实力背书

本剧中节目荣获多项中国杂技金菊奖、省级金奖等国内顶级赛事荣誉，多次登台春晚舞台。

节目矩阵：

高台跳水、大飞人、浪桥入水、四轴飞轮、环球飞车……

有速度，有高度，有水中惊险，有空中飞渡，有近距离沉浸，有全场环绕。

云台山有一台真正飞天入水的山水杂技秀。

体验强沉浸——身临其境

本演出通过舞台、影像、灯光、音乐、水效、雾效和观众区互动，形成沉浸式体验。

沉浸技术矩阵：

180度环绕影像、观众席光纤花盘、观众区灯光氛围联动、水雾向观众区延展、声音从四周涌来、高空动作越过观众视线、舞台与观众席边界被弱化

我们不堆设备，我们让每一台设备都上台演戏。让观众感到自己不是旁观者，而是误入云台秘境的一员。

问水云台，山水归心

《问水云台》不是一台普通演出。

它是云台山水的舞台化表达，

是河南杂技的时代化呈现，

也是一场关于相逢、守护、放下与归心的东方寓言。

我们希望让游客来到云台山，

不只看见山水，

更在山水之间，

看见自己。

一技承千年，一秀展云台。

山水相逢，心安云台。
